# **App Name**: AccountabilityOS

## Core Features:

- Role Selection: Role selection screen allowing the user to select one of several defined roles.
- Role-Based Dashboard: Dashboard display that is specific to the user's selected role.
- Sidebar Profile and Role Switcher: Sidebar to display profile information (name, role) and enable role switching.
- Static Usernames: Usernames remain consistent across sessions, so that a static identifier can be used.

- Secondary Accent: #D6A01D Gold – urgency, decision-making
- Background: #F9FAFB Soft white – clean, readable
- Card Background: #FFFFFF Panels and widgets
- Sidebar BG: #F3F4F6 Subtle grey
- Border/Divider: #E5E7EB Light grey lines
- Text Primary: #1F2937 Slate black – high contrast
- Text Secondary: #4B5563 Cool grey
- Highlight/Info: #0288D1 Calm blue for alerts
